 1단계 — 기초 이해 (Day 1 ~ 2)
────────────────────────────────────────

[ Day 1 ] 웹의 원리 이해 + 개발 환경 세팅
  분야: 이론 / 환경설정
  학습 내용:
  - 웹이란 무엇인가 — 브라우저가 HTML을 읽는 방법
  - VS Code 설치 및 기본 설정
  - 첫 HTML 파일 만들기 — "안녕하세요" 화면 출력
  - HTML 태그 기초: h1, p, a, img, div
  오늘의 결과물: 텍스트만 있는 센터 소개 1페이지


[ Day 2 ] CSS로 꾸미기 — 색상, 폰트, 레이아웃
  분야: CSS / 디자인
  학습 내용:
  - CSS란 무엇인가 — HTML과 CSS의 관계
  - 색상, 글꼴, 여백, 테두리 설정
  - Flexbox로 가로/세로 정렬하기
  - 따뜻한 톤의 색상 팔레트 선택 (복지센터 이미지)
  오늘의 결과물: 스타일이 입혀진 센터 소개 페이지


────────────────────────────────────────
 2단계 — 페이지 구조 설계 (Day 3 ~ 4)
────────────────────────────────────────

[ Day 3 ] 홈페이지 전체 구조 설계
  분야: 기획 / 구조설계
  학습 내용:
  - 대상별 콘텐츠 전략 — 보호자용 / 구직자용 분리
  - 메뉴 구성: 센터소개 / 서비스 안내 / 채용 / 자료실 / 문의
  - Navigation Bar (네비게이션) 만들기
  - Footer (하단 정보) 구성
  오늘의 결과물: 5개 메뉴가 있는 기본 레이아웃 완성


[ Day 4 ] 메인 페이지(홈) 제작
  분야: HTML / CSS / 디자인
  학습 내용:
  - 상단 히어로 영역 — 센터 슬로건 + 상담신청 버튼
  - 서비스 요약 카드 3개 (방문요양 / 목욕 / 인지활동)
  - 신뢰 지표 섹션 — 관리 어르신 수, 경력 연수 등
  - 모바일에서도 잘 보이도록 반응형 기초 적용
  오늘의 결과물: 완성된 메인 홈 페이지


────────────────────────────────────────
 3단계 — 핵심 콘텐츠 페이지 (Day 5 ~ 7)
────────────────────────────────────────

[ Day 5 ] 서비스 안내 페이지 — 보호자 대상
  분야: 콘텐츠 / 보호자
  학습 내용:
  - 장기요양등급 설명 (1~5등급 + 인지지원등급)
  - 서비스 신청 절차 시각화 (단계별 안내)
  - 급여 종류 안내 — 방문요양 / 방문목욕 / 방문간호
  - 자주 묻는 질문 (FAQ) 아코디언 구현
  오늘의 결과물: 보호자가 첫 방문 시 바로 이해할 수 있는 페이지


[ Day 6 ] 채용 페이지 — 요양보호사 구직자 대상
  분야: 콘텐츠 / 구직자
  학습 내용:
  - 센터 근무 환경 소개 — 차별화 포인트 강조
  - 근무 조건, 급여, 복리후생 안내
  - HTML Form 만들기 — 이름, 연락처, 경력 입력
  - JavaScript 기초 — 폼 입력값 확인 (유효성 검사)
  오늘의 결과물: 지원서 제출 가능한 채용 페이지


[ Day 7 ] 자료실 + 문의 페이지
  분야: 콘텐츠 / 기능
  학습 내용:
  - 자료실 — 복지 관련 PDF, 외부 링크 모음
  - 운영 정보 — 센터명, 주소, 전화번호, 운영시간
  - 카카오맵 API 연동 (지도 삽입)
  - 문의 폼 구성 (이름 / 연락처 / 문의내용)
  오늘의 결과물: 오시는 길 + 문의 완료 페이지

 4단계 — 완성 & 배포 (Day 8 ~ 10)

[ Day 8 ] JavaScript로 인터랙션 추가
  분야: JavaScript / UX
  학습 내용:
  - JavaScript란 무엇인가 — HTML/CSS와의 차이
  - 모바일 햄버거 메뉴 구현
  - 스크롤 시 헤더 고정 (sticky nav)
  - 상담 신청 버튼 → 전화연결 / 카카오 링크 연동
  오늘의 결과물: 살아 움직이는 인터랙티브 홈페이지


[ Day 9 ] SEO + 접근성 + 최종 점검
  분야: SEO / 최적화
  학습 내용:
  - 메타 태그 설정 — 구글 검색 결과에 잘 뜨도록
  - OG 태그 — 카카오톡/SNS 공유 시 미리보기 설정
  - 이미지 최적화 (용량 줄이기, alt 텍스트)
  - 스마트폰 / 태블릿 / PC 화면 최종 점검
  오늘의 결과물: 검색 최적화된 완성 홈페이지


[ Day 10 ] 인터넷에 올리기 — 실제 배포
  분야: 배포 / 도메인
  학습 내용:
  - GitHub Pages 또는 Netlify로 무료 배포
  - 도메인 연결 방법 안내 (선택 사항)
  - 구글 서치콘솔 등록 — 검색 노출 시작
  - 네이버 지도 / 카카오 채널 연동
  오늘의 결과물: 실제 인터넷에서 접속 가능한 홈페이지 완성!
